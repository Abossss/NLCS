# 基于自然语言处理的智能客服系统

## 项目简介
本项目是一个基于自然语言处理技术的智能客服系统，能够自动回答用户问题，提高客户服务效率。

## 功能特点
- 自然语言理解与处理
- 知识库自动检索
- 多轮对话管理
- 用户反馈学习

## 安装指南
1. 克隆项目仓库
2. 安装依赖：`pip install -r requirements.txt`
3. 初始化数据库：`python database/CreateDatabase.py`
4. 导入知识库数据

## 使用说明
1. 启动服务：`python main.py`
2. 访问 `http://localhost:5000`
3. 在聊天界面输入问题

## 开发环境
- Python 3.8+
- Flask 2.0+
- SQLite 3
